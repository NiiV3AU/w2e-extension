# w2e - Watch To Embed
Convert YouTube URLs from "/watch?v=xxx" to "/embed/xxx". 
# **"Why"** you ask?
The reason being is that the new **adblock popup on yt is hella annoying** and with just changing "watch" with "embed" in the video-url its **gone**!
